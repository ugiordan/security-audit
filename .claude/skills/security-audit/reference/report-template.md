# Report Templates

## Executive Report (default)

Shows critical + high findings only. Generated by `report.py`.

Sections:
1. **Header**: repo, date, branch, commit, tool count
2. **Executive Summary**: severity counts table
3. **Critical Findings**: detailed table with source, file, line, title, detected_by
4. **High Findings**: same format
5. **Tool Coverage**: tool name, finding count, top category
6. **Recommendations**: top 5 actionable items from critical/high findings

## Full Report (`--full`)

Everything in executive report plus:
- Medium Findings table
- Low Findings table  
- Info Findings table (if any)
- Cross-tool overlap analysis

## Report File Naming

- `executive-report.md` - default output
- `full-report.md` - when `--full` flag used
- `executive-report.docx` - when `--format docx` used
